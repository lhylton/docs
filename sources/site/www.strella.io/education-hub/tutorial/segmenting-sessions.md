How-to Guide

This tutorial walks you through how to segment your sessions by participant groups. Learn how to break out specific segments to uncover unique behaviors, attitudes, and insights, in order to get a clearer, more nuanced understanding of your research findings.

## Segmenting Sessions

When analyzing your research results, it can be valuable to break out specific participant segments - since different groups often surface unique behaviors, attitudes, or insights.

To segment participants, you have the ability to add session tags to participant sessions. You can do this by conducting the following steps:

1. Open an individual session
2. Click 'Add tags'
3. Type in a new tag and click enter, or select a previously created tag

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866db99f68e0f294a757e88_Screenshot%202025-07-03%20153539.png)

Once the tags are added to participant sessions, you can use the session tags on the responses tab. On the responses tab, select 'Filters' and click the 'Session Tags' filter from the drop down menu.

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866dc2f696ef35b0d93d7d1_Screenshot%202025-07-03%20153703.png)

Select the session tag for the segment you want to analyze further. Selecting this tag will filter down responses to include only those with the selected tag, and you can then use the chat feature or search bar.

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866dcc65a27fc0700bb24f5_Screenshot%202025-07-03%20153948.png)
