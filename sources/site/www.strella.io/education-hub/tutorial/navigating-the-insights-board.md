How-to Guide

The Saved tab is your personal insights hub - where the most important takeaways live. Whether you’ve starred AI-generated themes or added your own insights, this space keeps everything organized and easy to access. This tutorial will walk you through how to use the Saved Tab / Insights board so you and never lose track of what really matters.

## Navigating The Saved Tab / Insights Board

The saved tabis a repository of all the insights / key themes you have saved. These insight scan be saved in a number of ways:

- Selecting the star icon beside a key theme in the objective or question level analysis
- Using the chat feature and saving the insights generated from the chat
- Creating your own highlight reels / insights

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866e74f9d80efc4a7ea3465_Screenshot%202025-07-03%20162539.png)

You can click into each insight included in the saved tab to watch the associated highlight reel, read session notes, and understand overall sentiment for that particular insight.
