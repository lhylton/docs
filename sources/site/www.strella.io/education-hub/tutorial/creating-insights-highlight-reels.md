# Creating Insights & Highlight Reels

How-to Guide

This tutorial walks you through how to create insights and highlight reels using participant transcripts or the Chat With Your Research feature. Learn how to capture key moments and build compelling stories that bring your findings to life.

## Creating Insights & Highlight Reels

If you are watching participant sessions or reading through a transcript, and you find an insight you think is interesting or important, you can create your own highlight reel. To do so, conduct the following steps:

### Navigate to the part of the transcript where this insight was discussed

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866ebe1a3b74f18e2cbdbe9_Screenshot%202025-07-03%20164111.png)

### Click your mouse and drag across the part of the transcript where this insight was discussed

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866ec2986d74392394a55a5_Screenshot%202025-07-03%20164157.png)

### Click ‘Add note’ in the pop-up that appears

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866ec3b2ac2f85c73030514_Screenshot%202025-07-03%20164157%201.png)

### Click the 3 dots on the note that was just created on the right side of the page

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866ec6e5fc15ddb1218fa99_Screenshot%202025-07-03%20164404.png)

### Click ‘Add to insight’ and select an existing insight or create a new insight by typing an insight title and selecting it

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/6866ec9493f5745d5dead1d8_Screenshot%202025-07-03%20164436.png)

You can then repeat this process in each participant session where this insight was discussed.

The insight will be added to the ‘Saved’ tab where you will be able to watch the newly created highlight reel.

Another way to create a new highlight reel is by using the chat feature. If you’re using the chat feature and a particular insight generated is interested, click ‘Save insight’. This will save the insight to the ‘Saved’ tab and will automatically generate an associated highlight reel.

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/68700babf78c4fbfc3255026_Screenshot%202025-07-10%20145100.png)
