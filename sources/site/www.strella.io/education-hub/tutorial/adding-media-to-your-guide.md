# **Adding Media To Your Question Guide**

Researchers may choose to add media to a question guide to help test concepts, or watch how user interact with a website or app.

To add media to your question guide, click on ‘Add a step’ at the bottom of the panel on the left side. Scroll down through the different stimuli until you reach the various media types.

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/68cb141f7260d10d4ab3184d_image%20(16).png)

You can choose to include the following media types:

- Figma prototype
- Embed
- Website
- Image
- Video

To learn more about the different stimuli / media types you can add, visit the '[Setting up your question guide](https://www.strella.io/education-hub/tutorial/setting-up-your-question-guide)' tutorial and read through the 'Understanding question guide elements' section.

Let's say for example you wanted to insert a website link to watch participants navigate through it. Once you select 'Website' from the media options, paste the link to the website in the panel on the right side and select ‘Import website URL’:

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/68cb1550271da7966307f8f8_image%20(18).png)

Include tasks, questions and/or statements about the website by clicking on the three dots next to the website label, and clicking ‘Add step’.

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/68cb15727549b2082b1670e3_image%20(20).png)

You will see which questions, tasks, etc. will be asked while the website is open by looking at which steps are included within the website toggle.

![](https://cdn.prod.website-files.com/66d9ac37e276b11648312b1f/68cb1587dbd0700a0d1b6c0a_image%20(21).png)

You can follow similar steps for other media types.
